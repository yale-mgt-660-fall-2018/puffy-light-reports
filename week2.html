Narrative of the week

This week we had our first sprint planning meeting where we set up a first subset of user stories in trello Trello link

During the sprint planning meeting we agreed on the:
  1. Acceptance criteria
    1. Pull request performed (ideally should be code merged and promoted to Heroku but we want to break the dependency between the code writer and the code merger)
    2. Test running in local
    3. Code aligned with the specification
  1. Usage of poker sizing and the usage of the user story of reference 1 point (which we decided to be the button to go to about page)
  1. Assignment of user stories for first sprint

During this week we have also been able to deploy the software in Heroku RSVPeasy

The team is experincing technical difficulties to get started but we have done some progress this week as indicated in "Sprint Completed Stories". Special thanks this week go out to Griffin, who helped us getting started.
